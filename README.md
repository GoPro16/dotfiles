## Installation

+ Clone repo

```bash
git clone https://github.com/robmcguinness/dotfiles.git
```

+ Create a symbolic link for each dotfile in the cloned repo

``` bash
ln -s ~/dotfiles/fish/config.fish ~/.config/fish/config.fish
```
